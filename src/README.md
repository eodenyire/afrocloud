# src/

Application source code for the Africa Connect Cloud web console.

- `pages/`: route-level product domains (auth, onboarding, compute, storage, IAM, etc.).
- `components/`: reusable UI and domain components.
- `hooks/`: auth/workspace/polling and shared client hooks.
- `lib/`: control-plane and provider abstraction logic.
- `integrations/`: infrastructure connectors (Supabase).
- `test/`: unit/integration test setup.
