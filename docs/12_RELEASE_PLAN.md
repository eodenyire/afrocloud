## 🚀 Release Plan

### MVP (Phase 1)

- Control-plane schema + IAM
- Compute, storage, networking, databases, edge CRUD
- Audit logs + cost tracking
- Console integration + OAuth logins

### Phase 2

- Provider adapters (AWS/Azure/GCP/Oracle)
- Async provisioning engine + retries
- IaC templates + policy validation

### Phase 3

- CLI + SDK GA
- Multi-region HA control plane
- Compliance exports + billing integrations

### Phase 4

- Advanced AI-driven IaC generation
- Enterprise onboarding automation
- SLA dashboards and support tooling
